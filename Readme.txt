This project is able to scan ports of a certain host server, return the numbers of open ports and their service.In additon,a Reference OS version is provided(which is NOT always accurate,even proper).
The files included are:
GUI.java	(operating interface)
Port.java	(a class that store the message of one port)
PortScanner.java	(scan the ports in threads)
Storeinfile.java	(store the scan results in a file,the savepath is editable on operating interface)
Readfile.java	(read the message in the file and show it on operating interface)
isPortknown.java	(receive a Port and look through it in the port_list.txt)
port_list.txt	(store the general port number and its service)
